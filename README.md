## Overview

This repository contains the WinGet manifests for Still Systems tools. While we submit our packages to the official `microsoft/winget-pkgs` repository, this repo serves as the source of truth for our WinGet distribution.

## Installation / Quickstart

You can install our tools directly using the official WinGet client:

```powershell
winget install stillsystems.nexusv
```

## Usage

To update:

```powershell
winget upgrade stillsystems.nexusv
```
